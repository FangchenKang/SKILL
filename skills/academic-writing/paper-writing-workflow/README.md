# paper-writing-workflow

`paper-writing-workflow` 是一个论文写作过程指导 skill，用来安排学术论文从想法、提纲、片段、初稿、返修稿到完整成稿的推进顺序。

它适合中文社科论文、公共管理论文、政治学论文、社会学论文、法学论文、政策研究论文和跨学科治理研究论文。它的重点不是直接润色文字，而是先确认研究对象、研究问题、中心论点、目标期刊、学科归属、材料证据、目标字数和当前阶段，再按第零轮和四轮审查推进论文。

核心流程包括论文胚胎搭建、论证链条审查、概念理论框架审查、材料证据机制审查、语言润色和目标期刊适配。涉及期刊文风、文献综述、引用格式、Word 输出、数据分析等具体任务时，应配合其他对应 skill 使用。
