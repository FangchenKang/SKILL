## Writing Notes

Write the profile in the language requested by the user. If the manuscript is Chinese and the user does not request English, write the profile in Chinese.

Do not use the template headings as filler. Each section should contain journal-specific observations supported by the corpus.

When confidence is low, name what additional papers would improve the profile.
