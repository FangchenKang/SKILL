# Draft Review and Revision Template

Use this reference when reviewing or revising a manuscript with `journal_style_profile.md`.

## Required Review Logic

Every comment must be tied to the target journal profile. Avoid generic advice such as "make the argument clearer" unless the comment explains which journal habit the draft fails to match.

Assess five levels:

1. Problem construction.
2. Concept placement.
3. Argument structure.
4. Evidence-to-theory movement.
5. Conclusion boundary.

## Paragraph-Level Diagnosis

For each important section or paragraph, diagnose:

- Location.
- Current function in the manuscript.
- Journal-style issue.
- Revision direction.
- Revised version when requested.

Use short excerpts only when needed to identify the passage. Do not reproduce long passages from published papers or the user's draft unless the user asks for a full rewrite.

## Revision Rules

Preserve the author's research question unless the profile shows that the question must be reformulated to become publishable for the target journal.

Preserve core evidence. Reframe how the evidence is introduced, sequenced, and connected to theory.

Convert broad value claims into analytical claims. For example, replace policy importance with a statement of what explanatory problem, mechanism, or conceptual ambiguity the article addresses.

Convert factor lists into mechanism prose. Show the process through actors, institutions, resources, authority, information, rules, meaning, or interaction.

Control conclusions. Name contribution, implication, and limitation in proportion to what the manuscript has actually shown.

## Revision Output Scale

If the user asks for diagnosis only, do not rewrite the whole manuscript.

If the user asks for selected revision, rewrite the highest-impact paragraphs first: abstract, opening paragraphs, contribution paragraph, theoretical framework bridge, empirical transition paragraphs, and conclusion.

If the user asks for full revision, revise section by section and explain only the most important changes.
