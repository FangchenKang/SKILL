# journal-style-profiler

`journal-style-profiler` 是一个目标期刊文风剖面 skill，用来从 10 到 20 篇代表性论文中提取某一期刊稳定的学术表达习惯。

它关注的不是词频、句长或套话，而是期刊如何提出问题、安放概念、组织文献综述、展开理论框架、让材料进入论证、推进机制解释，以及如何控制结论分寸。

主要输出包括 `journal_style_profile.md` 和 `journal_style_review.md`。前者用于形成可复用的期刊写作 profile，后者用于审读用户草稿，指出符合期刊文风的地方、不符合的地方、需要改写的句段和改写后的版本。
