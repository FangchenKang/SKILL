# Skills

这个目录按用途分类存放 Codex skills。

## Academic Writing

- `academic-writing/paper-writing-workflow`: 学术论文从想法、提纲、片段、初稿到完整成稿的分轮写作流程 skill。
- `academic-writing/journal-style-profiler`: 从目标期刊代表性论文中提取期刊学术表达习惯，并用于审读和改写稿件的 skill。

## Scholar Profiles

- `scholar-profiles/scholar-profile-writer`: 将政管、法学、社会学等社科学者的原始材料整理为中文学者简介的 skill。
