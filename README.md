# SKILL

这个仓库用于存放个人 Codex skills。当前 skills 按用途分为学术论文写作和社科学者简介两类。

## 目录结构

```text
generated-style-profiles/
  社会学研究文风.md
  社会文风.md
skills/
  academic-writing/
    journal-style-profiler/
    paper-writing-workflow/
  scholar-profiles/
    scholar-profile-writer/
```

## Skills

`skills/academic-writing/journal-style-profiler` 用于从目标期刊代表性论文中提取期刊学术表达习惯，并用该 profile 审读和改写论文草稿。

`skills/academic-writing/paper-writing-workflow` 用于安排学术论文从想法、提纲、片段、初稿到完整成稿的分轮写作流程。

`skills/scholar-profiles/scholar-profile-writer` 用于将政治学、公共管理、法学、社会学等社科学者的原始材料整理为中文学者简介。

`generated-style-profiles/社会学研究文风.md` 是已经生成好的《社会学研究》期刊文风 profile，可供后续论文写作和草稿审读调用。

`generated-style-profiles/社会文风.md` 是已经生成好的《社会》期刊文风 profile，可供后续论文写作和草稿审读调用。
